Le multi-layer perceptron est un type de rseau de neurones.   • activation: fonction d'activation utilise par les neurones
  • alpha: pnalit permettant le lissage
  • solver: fonction d'attribution du poids des informations
  • learning_rate: facteur d'apprentissage pour les variations de poids
  • max_iter: le nombre maximal d'itrations
